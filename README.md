# SymbolicMath Bot

Телеграм бот, который умеет работать с полиномами: брать производную, интеграл, решать уравнения и раскрывать скобки.

## Что умеет

| Команда | Что делает | Пример |
|---------|-----------|--------|
| `/diff` | Производная | `3*x^2` → `6*x` |
| `/integrate` | Интеграл | `x^2` → `0.333*x^3 + C` |
| `/solve` | Решить уравнение | `x^2-4=0` → `2, -2` |
| `/expand` | Раскрыть скобки | `(x+2)*(x-3)` → `x^2 - x - 6` |

## Какие выражения понимает

Бот работает с полиномами. Можно использовать:

- `x`, `y`, `z` — переменные
- `+`, `-`, `*`, `/`, `^` — операции
- Скобки `( )`
- Целые и дробные числа

## Команды бота

### Математика
- `/diff [выражение]` — производная
- `/integrate [выражение]` — интеграл
- `/solve [уравнение]` — решение уравнения
- `/expand [выражение]` — раскрыть скобки

### Управление
- `/start` — начать
- `/help` — справка
- `/history` — история операций
- `/stats` — статистика
- `/last` — последний результат
- `/clear` — очистить историю
- `/cancel` — отменить
- `/menu` — показать меню
